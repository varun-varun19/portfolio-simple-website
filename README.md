Welcome to my portfolio website repository! This project is built using only HTML and CSS to provide a straightforward, fast, and visually appealing showcase of my skills, projects, and experience.

Overview
This website consists of two main pages:

Home Page: An introduction about me, covering my skills and a brief overview of my professional background.
Projects Page: A display of my key projects, with descriptions and any relevant links for further details.
Features
Simple Design: Focuses on clarity and readability, with a minimalist layout that emphasizes content.
Responsive Layout: Adapts to various screen sizes for a seamless experience on desktop and mobile devices.
Quick Load Times: Built with minimal CSS, ensuring fast loading for an optimal user experience.

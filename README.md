Features
Home Page:
Displays an introduction about me, my skills, and professional background.

Contact Page:
Includes a contact form and relevant links to connect with me.

This project was built as a simple yet effective way to share my portfolio online. It highlights my work and provides an easy way for potential collaborators or employers to get in touch.

CSS3: Styling and layout design.
